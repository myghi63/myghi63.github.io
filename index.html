My projects

Source-code and schematics are available at Gitlab/Github, but you can also see some Linux and Hardware related content at my youtube channel.

Database Backend for a Telegram Bot (@CookieMWbot)

Written in java, this backend is responsible for managing Cookiebot's database. Implemented using Spring Boot API.

Arch Linux Installer

This project contains scripts that automate the Arch Linux OS installation. It is currently deprecated in favor of a new installer (click here to visualize it).

Automatic Server Backup to a External Drive

This project allows anyone to backup the data from a Linux server to a encrypted external drive, by just plugging it in!

An Operational System Designed to Repair Computers

It's a Arch-based OS made to diagnose, recover data and run some tests to help fix computers!

Automatic PC to Server Sincronization and Snapshot Generation

Written in Bash script, it syncs data from my Linux PC to my home server via the local network. Then, the server creates snapshots (light copies) every night, separating them into dates to prevent data loss.

Personal Scripts for Everything!

This is a collection of scripts and configuration files to help me and my friends configure Linux stuff.

High Fidelity Power Amplifier

This is a amplifier that let me use a pair of bookshelf speaker boxes, with very high quality and two outputs of 38W. The Build log is planned to be published on youtube.

Low-Noise Condenser Microphone Amplifier

With a quality condenser microphone and this design, I can record and make calls with virtually no background noise other than the environment itself!
